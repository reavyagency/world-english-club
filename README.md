# World English Club 🌍🗣️

Plataforma do **World English Club** (Clube Mundial de Inglês) — projeto em fase inicial de desenvolvimento.

## 📋 Sobre

Repositório do projeto World English Club. A estrutura e as tecnologias serão definidas conforme o desenvolvimento avança.

## 🗂️ Estrutura do projeto

```plaintext
World-English-Club/
├── .agents/          # AG Kit — toolkit de agentes de IA (configurações, workflows, skills)
├── .gitignore
└── README.md
```

> A pasta `.agents/` contém o **AG Kit**, um conjunto modular de agentes especialistas, skills e workflows para auxiliar o desenvolvimento assistido por IA. Veja [.agents/ARCHITECTURE.md](.agents/ARCHITECTURE.md) para detalhes.
